# Script Standartlari

Bu bolum, script yazim ve dokumantasyon standartlarini tanimlar.

## Adlandirma
- Kisa ve amaca yonelik isim
- Fiil + nesne: `backup-db.sh`, `rotate-logs.sh`

## Dokumantasyon
- Ust kisimda amac, gereksinimler, kullanim ornegi
- Giris/CIKIS parametreleri net olmali

## Dogrulama
- Script calisinca bir durum mesaji vermeli
- Hata durumlari belirgin exit code uretmeli
